## Basic Chip
By default, a chip only requires the 'text' prop to be passed in. If no other props
are provided then the chip will be rendered with the default styling.

<Canvas withToolbar sourceState="shown" of={Stories.Default} />

## With Icon

To add an icon to the chip, pass the 'icon' prop. The icon prop should be a string (see the icon library for available icons).

<Canvas withToolbar sourceState="shown" of={Stories.WithIcon} />
